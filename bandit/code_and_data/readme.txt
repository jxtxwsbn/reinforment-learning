package required:

numpy
matplotlib

if you want to generate the data for each question, run the following code.

python testbed.py
python epsilon.py
python nonstationary.py
python oi.py
python ucb.py
python investigation.py

if you want to plot the data, please generate the data firstly and store the data in the 'data' foleder and
then run the following code:

python plot_violin.py (you could run this directly without generating the data)
python epsilon_plot.py
python nonstationary_plot.py
python oi_plot.py
python ucb_plot.py
python investigation_plot.py
